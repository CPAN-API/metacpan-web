Module Install Instructions

To install <% module.documentation or module.module.0.name %>, simply copy and paste either of the commands in to your terminal

CPANminus

				cpanm install <%  module.documentation or module.module.0.name %>

CPAN shell

				perl -MCPAN -e shell
				install <% module.documentation or module.module.0.name %>

For more information on module installation please visit The detailed CPAN module installation guide
